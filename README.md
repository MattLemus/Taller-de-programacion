# Ejercicios Capítulo 3 - Programación II

Este proyecto contiene la resolución de 5 ejercicios del capítulo 3 del libro **Cómo programar en Java** de **Deitel y Deitel**.

## Ejercicios incluidos

- **3.12** - La clase `Factura`
- **3.13** - La clase `Empleado`
- **3.14** - La clase `Fecha`
- **3.16** - `FrecuenciasCardiacas`
- **3.17** - `PerfilMedico`

## Estructura del proyecto

```text
src
 ├─ facturas
 │   ├─ interfaz
 │   │   └─ Main.java
 │   └─ negocio
